Introdução a Linguagem C

A linguagem C é o alicerce fundamental da computação moderna porque estabelece uma ponte direta entre a lógica humana e a execução física do hardware. Ao contrário de linguagens que escondem a complexidade através de abstrações, o C coloca o programador no controle total, permitindo a gestão precisa da memória e do processador.

É a escolha padrão para sistemas onde o desempenho é crítico: kernels de sistemas operacionais, drivers, sistemas embarcados e motores de jogos. Por ser compilada e tipada de forma estática, o código é transformado em linguagem de máquina, resultando em programas extremamente rápidos e leves. Além disso, sua sintaxe influenciou gigantes como C++, Java e JavaScript.

Estrutura e Sintaxe Fundamental

Todo programa em C possui uma anatomia padrão. A peça central é a função main, que indica ao sistema operacional onde a execução começa. O código é organizado entre chaves {} e cada instrução individual deve, obrigatoriamente, terminar com um ponto e vírgula ;.

Esqueleto do Código
#include <stdio.h>

int main() {
    // As instruções ficam aqui dentro
    printf("Olá Mundo!"); 
    return 0; // Indica que o programa terminou com sucesso
}

Bibliotecas: As Ferramentas Prontas

Bibliotecas são coleções de funções que evitam "reinventar a roda". São acessadas pela diretiva #include.

stdio.h (Standard Input-Output)

Gerencia a entrada e saída de dados. Essencial para ler o teclado e exibir informações no monitor através de funções como printf e scanf.

stdlib.h (Standard Library)

Lida com operações do sistema, gestão de memória dinâmica (malloc/free), geração de números aleatórios e controle de processos.

math.h (Mathematical Functions)

Oferece "superpoderes" matemáticos como sqrt (raiz quadrada), pow (potenciação) e funções trigonométricas.

string.h (String Handling)

Vital para manipular textos (vetores de caracteres), permitindo medir tamanhos (strlen), copiar (strcpy) ou comparar textos (strcmp).

Exemplo de Uso
#include <math.h>
// Agora o programa pode calcular raízes quadradas

Armazenamento: Variáveis e Tipos

No C, o computador reserva espaços fixos na memória. Cada tipo de dado exige um espaço diferente:

  • int: Números inteiros. Máscara de leitura/escrita: %d.
  • float: Números decimais simples. Máscara: %f.
  • double: Decimais de alta precisão (cálculos científicos). Máscara: %lf.
  • char: Um único caractere entre aspas simples ('A'). Máscara: %c.
  • Strings: Sequências de caracteres (textos) armazenadas como vetores. Máscara: %s.
Exemplo
int idade = 25;
float preco = 19.90;
char letra = 'C';

Operadores e Comparações

Os operadores realizam cálculos e avaliações lógicas. É crucial distinguir a atribuição da comparação:

  • = : Atribui um valor (guarda o dado).
  • == : Compara se dois valores são iguais.
  • && e || : Operadores lógicos "E" e "OU".
  • % : Operador de módulo (retorna o resto da divisão).

Estruturas Condicionais

O if avalia uma condição. Se for verdadeira, o bloco é executado; caso contrário, o fluxo pode seguir pelo else ou else if. O switch case serve para seleções rápidas entre múltiplas opções fixas.

Exemplo de Decisão
if (nota >= 7) {
    printf("Aprovado");
} else {
    printf("Recuperação");
}

Laços de Repetição (Loops)

Evitam a repetição manual de comandos. No C, cada laço possui uma aplicação ideal:

  • FOR: Ideal para quando se sabe o número exato de repetições (como um atleta dando 10 voltas em uma pista). Agrupa início, condição e incremento.
  • WHILE: Repete enquanto uma condição for verdade (como encher um reservatório enquanto o sensor for "não cheio"). Testa antes de executar.
  • DO WHILE: Executa o bloco ao menos uma vez e testa a condição no fim (ideal para menus de usuário).
Exemplo (FOR)
for (int i = 1; i <= 10; i++) {
    printf("Volta %d\n", i);
}

Funções e Recursão

Funções são blocos independentes que realizam tarefas específicas, facilitando a reutilização do código. A Recursão ocorre quando uma função chama a si mesma para resolver um problema menor (como o cálculo de um Fatorial).

Toda função recursiva exige um "caso base" (condição de parada) para evitar o travamento do sistema por estouro de memória.

Exemplo de Função
int somar(int a, int b) {
    return a + b;
}

Ponteiros e Memória

Ponteiros são variáveis que não guardam valores comuns, mas sim o endereço de memória de outra variável. Eles permitem manipular diretamente onde o dado está fisicamente no hardware.

  • & : Operador que extrai o endereço da memória.
  • * : Operador que acessa o valor guardado no endereço apontado.
Exemplo de Ponteiro
int valor = 10;
int *ptr = &valor; // ptr aponta para o endereço de 'valor'
printf("Endereço: %p | Valor: %d", ptr, *ptr);
